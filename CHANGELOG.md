# SAPDP Changelog

This file is historical release evidence. It is not protocol authority.

## v6.0.0

- Added mandatory, self-contained thread handoffs after Design Freeze and Materialization.
- Added Codex App deep links with deterministic CLI and IDE fallbacks.
- Added same-ref handoff validation, deterministic failure priority, and a mechanical renderer.
- Added a machine-readable Runtime Capsule Authority Registry location.
- Preserved staged loading, context budgets, and existing Materialization, Audit, and Release stdout.

## v5.0.0

- Added a bounded normative Runtime Capsule and stage-specific authority loading.

- Added repository-evolution closure governance and durable findings snapshots.
- Added deterministic context bundling, context budgets, dependency validation, and v4.0.0 compatibility evidence.

## v4.0.0

- Replaced single-file runtime authority with a root authority and registered Flow and Module authorities.
- Added repository architecture, directory governance, Authority Digest, and CAP ownership.
- Corrected Protocol Evolution naming, transitions, version semantics, audit binding, and recovery behavior.
- Added durable Design Freeze evidence and idempotent Materialization and Release operations.
- Standardized DNA as a module with design, runtime, and Add DNA flow authorities.
- Added executable architecture, compatibility, module, flow, and script validation.
