## v4.0.0

- Replaced single-file runtime authority with a root authority and registered Flow and Module authorities.
- Added repository architecture, directory governance, Authority Digest, and CAP ownership.
- Corrected Protocol Evolution naming, transitions, version semantics, audit binding, and recovery behavior.
- Added durable Design Freeze evidence and idempotent Materialization and Release operations.
- Standardized DNA as a module with design, runtime, and Add DNA flow authorities.
- Added executable architecture, compatibility, module, flow, and script validation.
