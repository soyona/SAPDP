# Notion Official Evidence Register

Retrieved: 2026-06-18

This register contains only factual observations from Notion's official Help Center. It is the evidence set used by `dna/library/NotionDNA/NotionDNA_v1.md`. No blogs, reviews, forums, AI-generated content, or personal opinions are included.

## E-001 — Intro to workspaces

- Official source: https://www.notion.com/help/intro-to-workspaces
- Observed evidence: Work in Notion takes place in a workspace. The interface is split into a sidebar and editor. The sidebar contains pages and databases, supports nested pages, and provides workspace tools. The editor contains created content, and its top controls recede after typing begins.

## E-002 — What is a block?

- Official source: https://www.notion.com/help/what-is-a-block
- Observed evidence: Page content such as text, images, and tables is represented as blocks. Pages are stacks of blocks. Blocks can transform into other block types and can be rearranged by drag and drop.

## E-003 — Navigate with the sidebar

- Official source: https://www.notion.com/help/navigate-with-the-sidebar
- Observed evidence: The sidebar supports nested pages without a stated depth limit, expandable hierarchy toggles, drag-and-drop page reorganization, search, recent pages, and workspace sections. It can be opened and closed through controls or a keyboard shortcut.

## E-004 — Intro to databases

- Official source: https://www.notion.com/help/intro-to-databases
- Observed evidence: Databases are collections of pages; each database item is a page. Items can have properties. The same data can be displayed through different views, and views can independently control layout, visible properties, filters, sorting, and grouping. Inline database controls remain hidden until hover.

## E-005 — Database properties

- Official source: https://www.notion.com/help/database-properties
- Observed evidence: Database properties provide typed context including text, number, select, status, date, formula, relation, rollup, person, file, checkbox, URL, timestamps, authorship, ID, and other supported types. Properties can be used to filter, sort, and search data.

## E-006 — Sharing & permissions settings

- Official source: https://www.notion.com/help/sharing-and-permissions
- Observed evidence: A page's Share control supports invitations, access inspection, permission changes, link copying, and web publishing. Pages can be shared with workspace members, groups, teamspaces, or guests, with assigned permission levels.

## E-007 — Collaborate in a workspace

- Official source: https://www.notion.com/help/collaborate-within-a-workspace
- Observed evidence: Multiple people can view and edit the same page or database concurrently. Avatars show where teammates are working. Edits, comments, and suggestions appear in real time. Pages and databases can be locked to reduce accidental structural changes.

## E-008 — Comments, mentions & reactions

- Official source: https://www.notion.com/help/comments-mentions-and-reminders
- Observed evidence: Notion supports page-level discussions, inline comments attached to selected text or blocks, mentions, replies, resolution and reopening, comment display options, and a page comment pane.

## E-009 — Style & customize your page

- Official source: https://www.notion.com/help/customize-and-style-your-content
- Observed evidence: Pages support Default, Serif, and Mono typography, a small-text option, normal or full-width content, text and block colors or highlights, icons, covers, backlinks, page discussions, and callout blocks.

## E-010 — Writing & editing basics

- Official source: https://www.notion.com/help/writing-and-editing-basics
- Observed evidence: The slash menu inserts content blocks. Text selection exposes styling and transformation controls. A block handle exposes actions and supports block movement. Columns can be formed by dragging blocks, and a blue guide indicates placement.

## E-011 — Keyboard shortcuts

- Official source: https://www.notion.com/help/keyboard-shortcuts
- Observed evidence: Notion documents keyboard shortcuts for creating, styling, selecting, navigating, and moving content, including opening search and creating comments.
