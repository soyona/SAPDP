# Purpose

Define the technical constraints that must be respected during product implementation.

This artifact exists to establish implementation boundaries before MVP planning and engineering execution.

This artifact must not define:

- Product Requirements
- Product Structure
- UX Design
- Visual Design
- Architecture Design
- Database Design
- API Design
- MVP Scope
- Build Plan
