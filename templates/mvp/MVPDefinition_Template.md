# Purpose

Define the minimum product scope required for validation.

This artifact exists to determine the smallest product that can validate the intended outcome.

This artifact must not define:

- Product Requirements
- Product Structure
- UX Design
- Visual Design
- Technical Design
- Task Decomposition
- Build Plan
- Implementation Details
