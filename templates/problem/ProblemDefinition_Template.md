# Purpose

Define and validate the problem worth solving.

This artifact exists to ensure that:

- The problem is clearly understood.
- The problem is worth solving.
- Future solution design remains anchored to the real problem.

This artifact must not define:

- Solution Design
- Product Design
- UX Design
- Visual Design
- Technical Design
- MVP Scope

These subjects belong to later lifecycle stages.
