# Start a New Project with SAPDP

## Prerequisites

You need:

```text
GitHub Repository

Codex

Access to SAPDP Repository
```

---

## Copy Into Codex

Open a new Codex session and paste:

```text
Initialize a new project using SAPDP.

Protocol:
https://github.com/xxx/SAPDP

Project Name:
<MyProject>
```

If Project Name is omitted:

```text
NewProduct
```

will be used as the default project name.

---

## Expected Result

Codex will:

```text
1. Read SAPDP Runtime

2. Generate Project Scaffold

3. Generate Bootstrap Files

4. Generate Workspace Files

5. Generate Runtime Files

6. Execute Bootstrap Audit

7. Enter Problem Stage
```

---

## Success Criteria

Bootstrap is successful when Codex reports:

```text
Bootstrap Audit Result

PASS
```

and:

```text
Current Stage:
Problem
```

---

## Next Step

Follow the guidance returned by Codex.

SAPDP will provide:

```text
Current Stage

Current Owner

Next Artifact

Next Action
```

Continue execution according to SAPDP.
